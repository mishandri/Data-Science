# Таргетированный маркетинг: прогнозирование отклика на кампании

Проект направлен на решение задачи бинарной классификации в маркетинге. Цель — предсказать, примет ли клиент хотя бы одно предложение из серии рекламных кампаний, на основе его демографических данных и истории покупок.

Это позволяет бизнесу снизить издержки на рассылку и повысить прибыль за счет таргетирования только потенциально заинтересованных клиентов.

## Постановка задачи

Даны два датасета:
- `train.csv` — обучающая выборка с историей ответов на 5 маркетинговых кампаний (`AcceptedCmp1-5`) и финальным полем `Response`.
- `test.csv` — тестовая выборка без целевого признака.

**Целевая переменная `Target`**:
- `1` — клиент принял хотя бы одно предложение (хотя бы одна из кампаний + Response = 1).
- `0` — клиент отказался от всех предложений.

Необходимо предсказать `Target` для всех записей в `test.csv` и сформировать файл с колонками `ID` и `Target`.

## Данные

- **Количество записей в train**: 1792
- **Количество записей в test**: 448
- **Признаки**: демография (возраст, образование, доход), семейное положение, наличие детей, история покупок в категориях, активность на сайте и т.д.
- **Пропуски**: присутствуют в поле `Income` (заполнены медианой).

## Этапы обработки

1. **Генерация целевой переменной**  
   `Target = OR(AcceptedCmp1, AcceptedCmp2, ..., AcceptedCmp5, Response)`

2. **Предобработка данных** (функция `prepare()`):
   - Кодирование категориальных признаков:
     - `Education` → числовая шкала (Basic → 0.1, PhD → 0.9)
     - `Marital_Status` → числовая шкала (Single → 0.0, Married → 0.2, ...)
   - Удаление поля `Dt_Customer` (дата регистрации)
   - Заполнение пропусков в `Income` медианой
   - Масштабирование всех признаков (кроме `ID`) в диапазон `[0, 1]` с помощью `MinMaxScaler`

3. **Моделирование**  
   Используется **Gaussian Naive Bayes** (`sklearn.naive_bayes.GaussianNB`) — простой и быстрый алгоритм, хорошо работающий с независимыми признаками.

4. **Формирование результата**  
   Сохраняется файл `result.csv` с колонками `ID` и `Target`.

## Используемые библиотеки

- `pandas` — работа с данными
- `numpy` — численные операции
- `scikit-learn`:
  - `MinMaxScaler` — нормализация
  - `GaussianNB` — классификатор
  - `train_test_split` (импортирован, но не использован в финальной версии)

```python
train['Target'] = (train['AcceptedCmp1'] + train['AcceptedCmp2'] +

```

2. Предобработка данных

Функция `prepare()` выполняет следующие шаги:

- Кодирование категориальных признаков:

- Заполнение пропусков в `Income` медианным значением

3. Моделирование

Используется **Gaussian Naive Bayes** (`sklearn.naive_bayes.GaussianNB`):

```python
nb = GaussianNB()
nb.fit(X_train, y_train)
result = nb.predict(X_test)
```
